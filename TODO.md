# Multi Platform Agent - 功能清单

## 已完成功能 ✅

| 功能 | 接口 | 说明 |
|------|------|------|
| GitLab MR 审查 | `POST /review/gitlab/{projectId}/{mrIid}` | 手动触发 MR 代码审查 |
| GitHub PR 审查 | `POST /review/github/{owner}/{repo}/{prNumber}` | 手动触发 PR 代码审查 |
| GitLab Webhook | `POST /webhook/gitlab` | 自动监听 MR open/reopen/update 事件 |
| GitHub Webhook | `POST /webhook/github` | 自动监听 PR opened/reopened/synchronize 事件 |
| 健康检查 | `GET /health` | 查看服务状态和 LLM 配置 |
| 代码质量审查 | - | 可读性、逻辑、复杂度、重复代码、最佳实践 |
| 安全漏洞扫描 | - | SQL 注入、XSS、命令注入、敏感信息泄露等 |
| 变更总结 | - | MR/PR 变更摘要生成 |
